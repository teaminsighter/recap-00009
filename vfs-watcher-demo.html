📋 How to Install & Use

  1. Install Tampermonkey extension in Chrome/Firefox
  2. Click Tampermonkey icon → Create new script
  3. Delete the default code
  4. Copy the entire content from vfs-slot-watcher.user.js
  5. Paste and save (Ctrl+S)
  6. Go to any VFS website - the panel will appear!
  7. Fill your details in 📝 Form Data tab
  8. Click 💾 Save Data
  9. Switch to 👁️ Watch tab
  10. Click ▶ Start to begin watching
  11. When slots appear → Sound plays → Form auto-fills → YOU click Book!
